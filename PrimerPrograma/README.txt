Tiempo programa secuencial 

Varia entre: 18 y 25 segundos en ejecutarse completamente.

Para medirlo se ocupó: time ./nombrePrograma


Ejecución del programa

Para ejecutar el programa se debe compilar primero de esta forma "g++ lab3SO_secuencial.cpp -o ex2" (el ex2 puede ser cualquiera) 
Luego de compilar se ejecuta directamente con "./ex2"
En caso de querer ver el tiempo antes se debe poner time antes de la ejecución.
